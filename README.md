Universal Grid Analyzer

Student Name:

Abdallah Zaza

Description:

This is a menu-driven Java program that works with a 2D array. The program allows the user to analyze the grid using different algorithms such as sums, maximum and minimum values, frequency counting, pattern detection, transformations, and validation.

Algorithms Implemented:

1. Display Grid
2. Row & Column Aggregation
3. Max & Min with Position
4. Frequency Counter
5. Pattern Detection (Increasing Sequence)
6. Grid Transformation
7. Subgrid Processing
8. Boundary & Diagonal Processing
9. Validation Algorithm
10. Menu System

Sample Output:

The program displays a menu in the terminal. The user selects an option and the result is shown (for example: grid display, sums, diagonals, etc.).

Challenges Faced:

Understanding nested loops and correctly working with row and column indices in a 2D array.
